# Декодирование HEX

Читает и декодирует из файла данные в формате **HEX** и записывает в бинарный файл.

## Назначение

https://toster.ru/

## Требования

* Visual Studio 2010 или новее
* .NET Framework 4.0

## Ключевые слова 

CSharp, C#, Windows Forms, HEX, Decode, System.IO
