# .NET

:::warning
This document has been translated using machine translation without human review.
:::

**.NET** is an open-source software platform (framework) for application development.
It allows you to create cross-platform applications.
It is supported by **Microsoft**.

As of 2025, **.NET** refers to **.NET Core**. The first version of **.NET Core** was released in 2016.

In the old days, **.NET** started with **.NET Framework** for **Windows**, the first version of which was released in 2002.
Now the **.NET Framework** platform is obsolete.
