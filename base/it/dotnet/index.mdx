# .NET

**.NET** — программная платформа (framework) с открытым кодом для разработки приложений.
Позволяет создавать кросс-платформенные приложения.
Поддерживается корпорацией **Microsoft**.

На 2025 год, под **.NET** подразумевается **.NET Core**. Первая версия **.NET Core** была представлена в 2016 году.

В старо-давние времена, **.NET** начинался с **.NET Framework** под **Windows**, первая версия которого была представлена в 2002 году.
Ныне платформа **.NET Framework** является устаревшей.
