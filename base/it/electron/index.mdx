# Electron

**Electron** (ранее известен как **atom shell**) — это программная платформа (framework) для разработки кросс-платформенных веб-приложений на базе **Node.js** и браузерного движка **Chromium**.

* [https://www.electronjs.org/](https://www.electronjs.org/)
* [https://electron-vite.org/](https://electron-vite.org/)
